Background Images, Amigos!

Properties

There are a few various background properties:

background-image
This propert is the location of the image itself, for exmaple: h1 {background-image: url('http://website.com/img/background.jpg');}
or

h1 {background-image: url('images/background.jpg'); }
This allows us to add colour to background. An example in CSS would be like this: h1 {background-color: green; }
